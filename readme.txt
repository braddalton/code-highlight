Style's The Code You Wrap In Pre Tags

== Description ==

This plugin uses the Google Code Prettify Script to Highlight & Style Code

The plugin adds styling to the existing <pre> Quick tag in your WordPress text editor

You can easily change the 12 colors in the plugins style sheet

And you can also add a color for alternate shading of lines

== Installation ==

1. Upload `plugin folder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Thats it. 

You can customize the styling of your code simply by changing the values for color in the plugins style.css file
